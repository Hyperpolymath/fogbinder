== Badge Design

Each badge features:

* **Left panel**: "RSR" label with dark background
* **Center**: Tier name with metallic gradient
* **Right circle**: Compliance percentage
* **Accessible**: Includes ARIA labels and title attributes
* **Clickable**: Links to canonical RSR specification

=== Colors

* **Rhodium**: Platinum/silver with purple accents (#B8B8D0 → #E8E8F0)
* **Gold**: Gold metallic gradient (#FFD700 → #DAA520)
* **Silver**: Silver/gray gradient (#C0C0C0 → #E8E8E8)
* **Bronze**: Bronze/copper gradient (#CD7F32 → #A0522D)
